# Task Tracker

FastAPI + Streamlit + PostgreSQL

## 🚀 Запуск

```bash
git clone https://github.com/Akds-ubuntu/task-tracker.git
cd task-tracker
docker-compose up --build
```
## 📍 Доступ локально

Frontend: http://localhost:8501

API Docs: http://localhost:8000/docs
